# Go Import Sample

```
1. new folder named "sample-app"
2. cli: go mod init sample-app
3. new folder as sample-app/pkg1
4. new file as sample-app/pkg1/f1.go
5. new folder as sample-app/pkg2
6. new file as sample-app/pkg2/f2.go
7. new file as sample-app/main.go
8. in f1.go, import sample-app/pkg2
```
